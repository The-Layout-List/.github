# [Layout List](https://laylist.pages.dev/)

This organization holds all the codebases used by the Layout List. Feel free to explore or create pull requests. 

## Repositories

**[website](https://github.com/The-Layout-List/website)**: The code and list data for [the Layout List website](https://laylist.pages.dev/). Based off [the TSL template](https://github.com/TheShittyList/GDListTemplate), with additional features developed by [sphericle](https://github.com/sphericle/).

**[bot](https://github.com/The-Layout/List/bot)**: A Discord bot allowing staff to easily interact with and update the data files for [the Layout List website](https://laylist.pages.dev/). Based off the [AREDL-ManagerV2](https://github.com/All-Rated-Extreme-Demon-List/AREDL-ManagerV2) bot, with additional features developed by [sphericle](https://github.com/sphericle/).


### In development
**[pointercrate](https://github.com/The-Layout-List/pointercrate)**: A fork of [Pointercrate](https://github.com/stadust/pointercrate), currently being adapted to be used as a backend server for the Layout List.
